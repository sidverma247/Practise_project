# Weather CLI

![Build](https://img.shields.io/badge/build-passing-brightgreen) ![License](https://img.shields.io/badge/license-MIT-blue) ![Version](https://img.shields.io/badge/version-0.1.0-orange)

A command-line tool that fetches current weather for any city.

## Installation

```bash
git clone https://github.com/you/weather-cli.git
cd weather-cli
pip install -r requirements.txt
```

## Usage

```
weather-cli --city "London"
```

```
weather-cli --city "Tokyo" --units metric
```
